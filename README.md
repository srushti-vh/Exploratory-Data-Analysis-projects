## Project 1: Wildfire Data Analysis Using R

### Overview
This project focuses on analyzing wildfire data stored in a SQLite database using R. The study involves cleaning, transforming, and summarizing data to uncover patterns related to wildfire size, causes, and occurrences across different states and years.

### Objectives
- Clean and preprocess wildfire data.
- Transform and summarize large datasets.
- Analyze wildfire occurrence trends across states.
- Study variations in fire size and causes.
- Generate visualizations to support data-driven insights.

### Tools Used
- R
- RStudio
- SQLite
- HeidiSQL
- dplyr
- ggplot2
- Quarto (.qmd)

### Key Tasks
- Data Extraction from SQLite Database
- Data Cleaning and Transformation
- Exploratory Data Analysis (EDA)
- Statistical Summarization
- Visualization of Trends and Patterns

### Visualizations Created
- Bar Charts
- Histograms
- Scatter Plots
- Box Plots
- Trend Analysis Charts

### Outcome
The analysis provided insights into wildfire patterns, including variations in fire size, causes, and occurrences across different states and years. The project demonstrates practical skills in data manipulation, database querying, and data visualization using R.

# Exploratory Data Analysis Projects

## Project 2: US Wildfire Data Analysis

### Overview
As wildfires continue to increase in frequency and impact, understanding their patterns and causes has become increasingly important. This project uses the US Wildfire Dataset to perform Exploratory Data Analysis (EDA) and generate meaningful insights through data visualization.

### Objectives
- Analyze wildfire occurrence patterns.
- Identify trends and relationships within the data.
- Study the distribution of wildfire incidents.
- Transform raw wildfire data into meaningful visual insights.

### Tools Used
- R
- RStudio
- SQLite
- HeidiSQL
- ggplot2
- Quarto (.qmd)

### Visualizations Created
- Scatter Plots
- Histograms
- Box Plots
- Correlation Matrix
- Additional exploratory charts

### Key Activities
- Data Cleaning and Preparation
- Data Exploration
- Trend Analysis
- Correlation Analysis
- Data Visualization

### Outcome
The project successfully transformed raw wildfire data into informative visualizations, helping to better understand wildfire patterns, distributions, and potential relationships between variables.
